Semantic Search Tool To Aid with Study for IFOA Exams


The project involved several key stages: firstly, a Python script was developed to scrape questions from past exam papers of both CP1 and CA1 from the IFOA and IAI, resulting in a dataset of about 3,000 questions. Subsequent data cleansing was performed to ensure data quality. Each question was then embedded using an open-source model hosted on Hugging Face, leveraging the HF API to avoid the need for local high-performance hardware. With a dataset containing both questions and embeddings, a semantic search tool was created, enabling users to input keywords related to a chapter and retrieve the most relevant questions based on cosine similarity. This tool proved invaluable in compiling study notes for the CP1 exam, showcasing the potential of AI-driven tools in educational contexts.

Skills used:

  • Python - Pandas, PyXL, PyTorch
  • Machine Learning

Skills learned:

  • Large Language Models
  • HuggingFace API Integration
